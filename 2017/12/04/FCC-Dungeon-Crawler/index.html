FCC Dungeon Crawler

2017-12-04


Links: Finished App | Code

Purpose

The classic game of Dungeon Crawler made with React. Beat the enemies to gain experience, pick up stronger weapons, pick up health packs to stay alive, and finally take care of the boss!

It was created as a project for freeCodeCamp. The full assignment definition can be found here.

Challenging Aspects

Game play algorithm and tuning. In terms of the use of React, it isn’t much different from the Game of Life. However, in order to make the game playable and enjoyable, a lot of thoughts has to go into setting the parameters. This is something I’d like to keep coming back to and make it more fun to play. I hope to add more maps too!

Thanks for checking it out.

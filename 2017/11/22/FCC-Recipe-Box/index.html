FCC Recipe Box

2017-11-22


Links: Finished App | Code

What It Does

It’s a recipe collection app where the user can view, create, edit, or delete recipes. The data is stored in the browser’s localStorage Object.

It was created as a project for freeCodeCamp. The full assignment definition can be found here.

Challenging Aspects

This was my first attempt at creating a modal in React. A quick Google search led me to a lot of complicated solutions that require extra packages and libraries. It all seemed like overkill, so in the end I simply triggered the CSS classes using React State.

The design is a bit cheap looking, but I think it’s clean, minimalist and intuitive. If I learn to utilize cloud storage to sync different devices… I might actually want to use it!

Thanks for checking it out.

FCC Voting App

2017-11-02


Links: Finished App | Code

Purpose

This was a full stack app project for freeCodeCamp, using the MERN stack (MongoDB/Express/React/Node).

Basically it allows the user to create/edit polls or vote on existing ones.
Some functionalities must be limited to authenticated users only.

The assignment definition can be found here.

Key Features

The most challenging tasks for this project were:

  • Complex routing with Express
  • Handling of poll database using MongoDB and Mongoose
  • Rendering with React. First-time experience with React. Did it without JSX.
  • Data visualization (pie chart) using D3.
  • Authentication with Passport.js. Final code has GitHub strategy only, but also tried local strategy.

Most of all, putting all of it together was a huge challenge and I got stuck many, many times. But it really helped me to understand how all the pieces come together. It was really a big confidence-booster for me.

Thanks for checking it out!

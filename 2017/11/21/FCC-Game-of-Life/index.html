FCC Game of Life

2017-11-21


Links: Finished App | Code

Purpose

This is my version of John Horton Conway’s Game of Life. It is intended to simulate dividing cells. Check out the link above for the detailed rules.

It was created as a project for freeCodeCamp. The full assignment definition can be found here.

Challenging Aspects

Rendering performance. Making sure that there is nothing unnecessary in each rendering cycle for the board.

Enjoyable Features. It wasn’t required in the assignment, but I added the ability to save the board, as well as the possibility to start from a few interesting famous patterns.

It’s a simple-looking game, but it is quite addicting. My kids love it!
